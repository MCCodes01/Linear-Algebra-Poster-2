Abstract

Linear transformations are the fundamental bridge between vector spaces, providing a powerful framework for understanding how vectors can be mapped while preserving their algebraic structure. This poster explores how these special functions enable us to represent complex operations as simple matrix multiplications, revolutionizing fields from computer graphics to quantum mechanics.

1. What Are Linear Transformations

Definition: A linear transformation T: V → W is a function between vector spaces that preserves vector addition and scalar multiplication:

1. T(u + v) = T(u) + T(v) for all vectors u, v in V

2. T(cv) = cT(v) for all vectors v in V and scalars c

Linear transformations map straight lines to straight lines and preserve the origin. They can be visualized as "stretching," "rotating," "projecting," or "reflecting" vector spaces.

If we define T: ℝ² → ℝ² as a 90° counterclockwise rotation, then:

T([1, 0]) = [0, 1]

T([0, 1]) = [-1, 0]

This is linear because rotation preserves vector addition and scalar multiplication.

Key Properties:

  • A linear transformation is completely determined by its action on a basis
  • Every linear transformation between finite-dimensional spaces can be represented by a matrix
  • The composition of linear transformations is also a linear transformation
  • Linear transformations preserve linear independence and span

2. Matrix Representation

Every linear transformation T: ℝⁿ → ℝᵐ can be represented by an m × n matrix A where:

T(x) = Ax for any vector x in ℝⁿ

To find the matrix representation:

  1. Choose a basis for the domain space V: {v₁, v₂, ..., vₙ}
  2. Compute T(v₁), T(v₂), ..., T(vₙ)
  3. Express each T(vᵢ) in terms of the basis of the codomain space W
  4. The coordinates of T(vᵢ) form the i-th column of the matrix

For the 90° rotation T: ℝ² → ℝ² mentioned earlier:

T([1, 0]) = [0, 1] and T([0, 1]) = [-1, 0]

So the matrix is: [0 -1; 1 0]

Important: The same linear transformation can have different matrix representations depending on the chosen bases for the domain and codomain.

3. Homework Example: Eigenvalues & Transformations

From Homework, Problem 2(a): Finding eigenvalues and eigenvectors of a transformation matrix.

Consider the matrix A representing a linear transformation:

A = ⎡ -4 2 ⎤ ⎣ 3 -9 ⎦

To find eigenvalues, we solve det(A - λI) = 0:

det(A - λI) = (-4-λ)(-9-λ) - 6 = λ² + 13λ + 30 = 0

Factoring: (λ+3)(λ+10) = 0

Eigenvalues: λ = -3, -10

This example demonstrates how eigenvalues reveal the fundamental behavior of a linear transformation - in this case, the transformation compresses space in the direction of both eigenvectors (negative eigenvalues), with more compression along the λ = -10 eigenvector.

4. Lab Example: Radiographic Transformation

From Lab 6: Analyzing radiographic imaging as a linear transformation.

Consider this radiographic setup where a 2×2 grid of voxels is mapped to radiograph readings:

T = ⎡ 1 1 0 0 ⎤ ⎢ 0 0 1 1 ⎥ ⎢ 0 1 0 1 ⎥ ⎣ 1 0 1 0 ⎦

This transformation matrix T maps the densities of voxels [x₁, x₂, x₃, x₄] to radiograph readings [b₁, b₂, s₁, s₂] from different angles (0° and 90°).

Question: Are any nonzero objects invisible to this operator?

An object is invisible if T(x) = 0 for x ≠ 0.

The solution shows that vectors in the form x = [x₁, -x₁, -x₁, x₁] are invisible to this transformation - they produce zero radiograph readings despite having nonzero densities!
